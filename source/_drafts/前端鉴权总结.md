我们在访问网站时通常会遇到登录的问题，前端鉴权方案主要有以下几种：

1. HTTP Basic Authentication：现在用的较少，主要用在一些较老的项目中。
2. session-cookie：依赖 cookie,只适用于 web 系统。
3. token: 目前主流鉴权方式，适用于多种客户端（浏览器、app 等）。
4. OAuth：第三方授权，方便快捷是以后的趋势。
