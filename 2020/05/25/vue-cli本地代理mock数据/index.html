vue-cli 4.0+版本配置

4.0+版本的vue-cli配置藏的更深，需要找到node_modules/@vue/cli-service/options.js文件中的devServer对象进行配置，配置方法同上面3.0版本类似。另外我们还可以在项目根目录自己新建一个vue.config.js配置文件，新建devServer对象利用before(app)或者proxy来进行请求代理。
